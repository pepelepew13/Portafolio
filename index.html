Unidad 1

Introducción

La investigación de operaciones aplica métodos analíticos y modelos matemáticos para apoyar decisiones óptimas en contextos reales. Se enfoca en usar mejor recursos limitados (tiempo, dinero, materiales) mediante la formulación y solución de problemas de optimización.

¿Qué resuelve?

  • Logística y cadenas de suministro
  • Producción y control de inventarios
  • Transporte y localización
  • Finanzas y portafolios
  • Salud y servicios
  • Telecomunicaciones y redes

Herramientas: programación lineal, entera, no lineal; teoría de colas; simulación; grafos y redes.

Fundamentos

Programación Lineal

La PL modela problemas con una función objetivo, variables de decisión y restricciones lineales. Permite elegir la mejor combinación de decisiones dentro de una región factible.

Estructura del modelo

Función Objetivo: Max/Min Z = c₁x₁ + c₂x₂ + ... + cₙxₙ Restricciones: aᵢ₁x₁ + aᵢ₂x₂ + ... + aᵢₙxₙ ≤/=/≥ bᵢ No negatividad: xⱼ ≥ 0

Áreas de aplicación

  • Planeación de producción
  • Mezcla de recursos
  • Asignación de personal
  • Ruteo y distribución
  • Presupuestos
  • Portafolios de inversión

Modelado

Ejemplo del herrero

Con 80 kg de acero y 120 kg de aluminio se fabrican bicicletas de paseo (X₁) y de montaña (X₂). A continuación, la formulación.

Variables y objetivo

Variables
X₁: # de bicicletas de paseo
X₂: # de bicicletas de montaña
Función objetivo (beneficio)
Max Z = 20.000·X₁ + 15.000·X₂

Restricciones e interpretación

Restricciones
Acero: 1·X₁ + 2·X₂ ≤ 80
Aluminio: 3·X₁ + 2·X₂ ≤ 120
No negatividad: X₁, X₂ ≥ 0
Lectura

La solución óptima indica cuántas unidades producir de cada tipo usando eficientemente los recursos.

Resolución

Método algebraico y variables de holgura

Para resolver por álgebra, llevamos las restricciones a igualdades agregando variables de holgura (sᵢ) que representan recursos sobrantes. Luego evaluamos la función objetivo en vértices.

Forma estándar

1·X₁ + 2·X₂ + s₁ = 80
3·X₁ + 2·X₂ + s₂ = 120
X₁, X₂, s₁, s₂ ≥ 0

Interpretación de holguras

Si s₁ > 0, sobró acero; si s₁ = 0, el acero es recurso crítico. Ayuda a identificar cuellos de botella.

Algoritmo

Método Simplex

Procedimiento iterativo que recorre vértices de la región factible, mejorando Z hasta alcanzar el óptimo.

1) Preparación

Escribir el problema en forma estándar, agregar holguras y construir la tabla inicial.

2) Variable de entrada

Elegir la columna con mayor potencial de mejora (coeficiente más negativo en Z si maximizamos).

3) Variable de salida

Aplicar la prueba de razón mínima para mantener factibilidad.

4) Pivoteo

Normalizar la fila pivote y anular el resto de la columna. Repetir hasta que no haya mejoras.

5) Lectura de solución

Las variables básicas dan los valores óptimos; Z indica el valor máximo/mínimo alcanzado.

Cierre

Conclusión

La programación lineal ofrece un marco riguroso para decidir con eficiencia. El flujo general es:
1) abstraer el problema
2) formular el modelo
3) resolverlo (álgebra o simplex)
4) interpretar resultados y holguras
5) comunicar conclusiones.

Este portafolio resume los conceptos y deja plantillas para replicar el proceso en nuevos problemas.
